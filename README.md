# ADOS Extension para pyRevit

Extension profesional de herramientas BIM para Autodesk Revit.

## Modulos incluidos

- **Aceros** — 9 herramientas: Cimientos, Columnas, Vigas, Zapatas, Estribos
- **Documentacion** — 4 herramientas: Acotado, Generacion de planos, Secciones
- **General** — Eliminar habitacion
- **Modelado** — 6 herramientas: Acabados de muros y suelos

## Instalacion rapida

1. Descarga `ADOS_Installer.exe` desde [Releases](https://github.com/fredyandresc97-art/ADOS-Extension/releases/latest)
2. Ejecuta el instalador y haz clic en **Instalar**
3. Reinicia Autodesk Revit

## Requisitos

- Autodesk Revit 2022 / 2023 / 2024 / 2025
- [pyRevit](https://github.com/pyrevitlabs/pyRevit/releases/latest) instalado
